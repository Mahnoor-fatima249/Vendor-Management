# Vendor & Quotation System

A full-stack web application designed to streamline procurement processes by centralizing vendor management and quotation tracking. This project was developed as part of the **Teyzix Core Internship (June Batch)**.

## 🚀 Overview

The Vendor & Quotation System replaces manual processes (like emails and spreadsheets) with a clean, responsive interface. It allows businesses to add vendors, request quotations, and compare them effectively to identify the most cost-effective solution.

## ✨ Key Features

* **Vendor Management:** Create, view, update, and delete vendor information.
* **Quotation Management:** Request quotations, assign them to specific vendors, and track status.
* **Real-time Dashboard:** Visualize system health with total vendors, active/pending quotations, and recent activities.
* **Quotation Comparison:** Automatically identify and highlight the most cost-effective quotation for better decision-making.
* **Responsive UI:** Built with a user-friendly interface that works across devices.

## 📋 Project Workflow

1. **Vendor Registration:** Add vendors via the "Vendors" tab.
2. **Quotation Request:** Submit requests in the "Quotations" tab by selecting a registered vendor.
3. **Comparison & Approval:** View the live comparison on the "Dashboard" and update the status (Approve/Reject).
